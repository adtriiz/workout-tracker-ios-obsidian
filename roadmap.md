# Project Roadmap

The following features are planned for future versions of the Obsidian Workout Tracker.

## Performance & Analytics
- [ ] **1RM Estimation:** Implementation of Brzycki/Epley formulas for strength tracking.
- [ ] **Volume Heatmaps:** Visualization of muscle group focus over time.
- [ ] **PR Alerts:** Notification when a personal record is broken during a session.

## Advanced Logging
- [ ] **Drop Sets:** UI support for reducing weight within a single exercise entry.
- [ ] **Plate Calculator:** Assistant to help calculate bar loading.
- [ ] **RPE Tracking:** Rate of Perceived Exertion logging for autoregulation.

## Integration
- [ ] **Direct Vault Sync:** Investigating deeper iOS integration for background writing.
- [ ] **Apple Health Sync:** Exporting workout duration and heart rate data.
- [ ] **Media Attachments:** Ability to link form-check videos or photos to logs.
